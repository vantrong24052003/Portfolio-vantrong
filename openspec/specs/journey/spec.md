## ADDED Requirements

### Requirement: Interactive Career Timeline
A vertical timeline (desktop) or horizontal timeline (mobile) allowing users to explore career milestones through interactive nodes.

#### Scenario: Milestone Navigation
- **WHEN** a user clicks/hovers on a timeline node (e.g., 2021, 2023, 2026)
- **THEN** the detail panel should update to show the corresponding organization, role, and achievements.

#### Scenario: Visual Feedback
- **WHEN** a node is active
- **THEN** it should feature a glowing Ruby Red effect and a ping animation.

#### Scenario: Detail Presentation
- **WHEN** a milestone is selected
- **THEN** it should display a glassmorphism card containing the timestamp, title, description, and technical stack used during that period.
